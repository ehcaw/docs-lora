## Prerequisites

Before you begin using Pointer, ensure you have the following prerequisites in place to guarantee a smooth and enjoyable experience:

<Info>
  **System requirements:**
  - A modern web browser (Chrome, Firefox, Safari, or Edge with recent updates)
  - A stable internet connection for real-time collaboration features
  - For local development: Node.js 20 or higher, npm or bun package manager
</Info>

## Step 1: Access Pointer

Pointer is designed to be accessible immediately without any complex setup process. Simply navigate to the Pointer application in your web browser to get started:

<Steps>
  <Step title="Open the application">
    Navigate to the Pointer application URL in your web browser. You'll be greeted with an elegant login page featuring classical artwork and a warm, inviting design that sets the tone for a focused, thoughtful note-taking experience.
  </Step>
  <Step title="Create an account or sign in">
    Pointer uses Clerk for authentication, providing you with multiple secure sign-in options including email/password, Google, GitHub, and other popular identity providers. Choose your preferred method and complete the authentication process to access your personal workspace.
  </Step>
  <Step title="Explore your workspace">
    Once authenticated, you'll be automatically redirected to your main workspace where you can begin creating and organizing your notes, whiteboards, and tasks.
  </Step>
</Steps>

## Step 2: Create your first note

Creating a new note in Pointer is intuitive and straightforward, designed to get you writing as quickly as possible:

<Steps>
  <Step title="Navigate to the notes section">
    In the main workspace, you'll see a sidebar on the left containing your folder structure and navigation options. This is your command center for accessing all your notes and organizing your content hierarchically.
  </Step>
  <Step title="Create a new note">
    Click the "New Note" button or use the keyboard shortcut to create a fresh document. A new note will appear in your current folder location, ready for you to start writing immediately.
  </Step>
  <Step title="Give your note a name">
    Click on the note title at the top of the editor to give your note a meaningful name. Good naming conventions will help you find your notes quickly as your collection grows over time.
  </Step>
  <Step title="Start writing">
    Begin typing in the editor area. The TipTap-powered editor supports rich text formatting including:
    
    - **Bold**, *italic*, and ~~strikethrough~~ text
    - Multiple heading levels for document structure
    - Bulleted and numbered lists
    - Code blocks with syntax highlighting
    - Tables for structured data
    - Images and media embeds
    - Links to external resources
  </Step>
</Steps>

## Step 3: Organize with folders

As your note collection grows, organizing your content into folders becomes essential for maintaining productivity and finding information quickly:

<Steps>
  <Step title="Create a folder">
    Right-click in the sidebar or use the folder creation button to create a new folder. Folders can be nested to any depth, allowing you to create a hierarchical structure that mirrors how you think about your knowledge domains.
  </Step>
  <Step title="Move notes into folders">
    Drag and drop notes from one location to another, or right-click and select "Move" to relocate notes into specific folders. The folder structure updates immediately across all your devices.
  </Step>
  <Step title="Navigate your structure">
    Expand and collapse folders in the sidebar to navigate through your content hierarchy. Click on any note to open it in the editor. The currently selected note is highlighted in the sidebar for easy reference.
  </Step>
</Steps>

## Step 4: Explore the rich text editor

The Pointer editor is built on TipTap, providing a powerful and flexible writing environment. Take some time to explore its capabilities:

### Formatting toolbar

A floating toolbar appears when you select text, providing quick access to common formatting options:

- **Bold** (Ctrl/Cmd + B)
- **Italic** (Ctrl/Cmd + I)
- **Underline** (Ctrl/Cmd + U)
- **Strikethrough**
- **Code** formatting
- **Highlight** with multiple colors
- **Text alignment** options
- **Link** insertion

### Slash commands

Type `/` at the beginning of a line to access the slash command menu, which provides quick access to:

- Different heading levels
- Bullet and numbered lists
- Code blocks with language selection
- Tables
- Images
- Horizontal rules
- And more formatting options

### Keyboard shortcuts

Pointer supports extensive keyboard shortcuts for power users:

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + B` | Bold |
| `Ctrl/Cmd + I` | Italic |
| `Ctrl/Cmd + U` | Underline |
| `Ctrl/Cmd + K` | Insert link |
| `Ctrl/Cmd + Z` | Undo |
| `Ctrl/Cmd + Shift + Z` | Redo |

## Step 5: Enable collaboration

One of Pointer's most powerful features is real-time collaboration. To share a note with others:

<Steps>
  <Step title="Open the share dialog">
    Click the share button in the note header or use the keyboard shortcut to open the sharing dialog. This dialog allows you to manage who has access to your document.
  </Step>
  <Step title="Add collaborators">
    Enter the email addresses of the people you want to share the note with. They'll receive an invitation to view and edit the document with you.
  </Step>
  <Step title="Enable collaborative mode">
    Toggle the collaborative mode switch to enable real-time synchronization. When enabled, all collaborators will see each other's changes instantly as they type.
  </Step>
  <Step title="Start collaborating">
    Once collaborative mode is enabled and your collaborators have joined, you'll see their cursors and selections in the document. Changes sync in real-time with no manual save required.
  </Step>
</Steps>

## Step 6: Try the whiteboard

For visual thinking and diagramming, Pointer includes an integrated whiteboard powered by Excalidraw:

<Steps>
  <Step title="Open the whiteboard">
    Navigate to the Whiteboard section from the main navigation. Your personal whiteboard is automatically created when you first access this feature.
  </Step>
  <Step title="Start drawing">
    Use the toolbar on the left to select different drawing tools:
    - **Selection** tool for moving and resizing elements
    - **Rectangle**, **ellipse**, **diamond**, and **arrow** shapes
    - **Line** and **freehand** drawing tools
    - **Text** tool for adding labels
  </Step>
  <Step title="Customize your drawings">
    Select any element to access customization options including stroke color, background color, stroke width, and more. Create professional-looking diagrams with just a few clicks.
  </Step>
  <Step title="Save automatically">
    Your whiteboard content saves automatically as you work. Close and reopen the whiteboard at any time—your work will be exactly as you left it.
  </Step>
</Steps>

## Step 7: Manage your tasks

Keep track of your to-dos with Pointer's integrated task management:

<Steps>
  <Step title="Open the tasks view">
    Navigate to the Tasks section from the main navigation to see your task dashboard.
  </Step>
  <Step title="Create a task">
    Click the "Add Task" button to create a new task. Provide a task name and optionally add:
    - **Description** for additional context
    - **Category** for organization
    - **Due date** for deadline tracking
    - **Tags** for flexible categorization
  </Step>
  <Step title="Manage tasks">
    Click the checkbox next to any task to mark it as complete. Edit tasks by clicking on them, delete tasks you no longer need, and filter your task list by category or completion status.
  </Step>
</Steps>
