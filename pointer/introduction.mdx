## What is Pointer?

Pointer is a full-featured note-taking and productivity application designed as an alternative to Notion, built with the goal of providing all the functionality you'd expect from a modern workspace app while avoiding subscription fees and adding genuinely useful AI capabilities. It combines the power of a rich text editor with real-time collaboration, whiteboard functionality, and intelligent writing assistanceâ€”all wrapped in a native desktop application.

The project was born from a simple frustration: paying $10/month for notes feels excessive. Rather than just accepting the cost, Pointer was built to demonstrate that powerful productivity tools can be created with modern web technologies and delivered as desktop applications with full functionality.

## Why Pointer exists

Modern productivity applications like Notion have set high expectations for what a workspace tool should offer:

- **Rich content editing** with blocks, databases, and embeds
- **Real-time collaboration** with presence awareness
- **Cross-device sync** for seamless access anywhere
- **Organization tools** like nested pages and search

However, these tools often come with significant drawbacks:

- **Subscription costs** that add up over time
- **Limited AI features** unless you pay even more
- **Browser dependency** for desktop usage
- **Privacy concerns** with cloud-only storage

Pointer addresses these concerns by providing:

## Core capabilities

### Rich text editing

Pointer uses Tiptap, a headless editor framework, to provide a powerful editing experience:

- **Block-based editing**: Create and organize content in blocks
- **Rich formatting**: Headings, lists, code blocks, tables, and more
- **Slash commands**: Quick access to formatting and content types
- **Markdown support**: Write in markdown if you prefer
- **Image support**: Embed and manage images within notes

### Real-time collaboration

Built on Convex for real-time functionality:

- **Multi-user editing**: Work on documents simultaneously
- **Presence indicators**: See who's viewing and editing
- **Conflict resolution**: Automatic merging of concurrent edits
- **Share permissions**: Control who can view and edit

### Knowledge graph

Visualize connections between your notes:

- **Automatic linking**: Connections based on content relationships
- **Interactive graph view**: Explore your knowledge visually
- **Tag exploration**: Navigate by tags and categories
- **Timeline view**: See your notes chronologically

### Whiteboard

Integrated drawing and diagramming:

- **Freehand drawing**: Sketch ideas naturally
- **Shape tools**: Create diagrams and flowcharts
- **Export capabilities**: Save whiteboards as images
- **Note integration**: Link whiteboards to notes

### Desktop experience

Wrapped in Tauri for native performance:

- **Fast startup**: Native application speed
- **System integration**: OS notifications, file access
- **Offline capability**: Work without internet
- **Small footprint**: Much lighter than Electron

## Technology foundation

Pointer is built on a modern, robust technology stack:

| Layer | Technology | Purpose |
|-------|------------|---------|
| **Editor** | Tiptap | Rich text editing with extensions |
| **Frontend** | Next.js, React | User interface and routing |
| **Backend** | Convex | Real-time data sync and storage |
| **Desktop** | Tauri | Native desktop wrapper |
| **AI** | Vercel AI SDK | Writing assistance features |
| **Styling** | Tailwind CSS | Consistent, responsive design |
| **Collab** | PartyKit, Y.js | Real-time collaboration infrastructure |
