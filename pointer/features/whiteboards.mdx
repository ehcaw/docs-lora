## Overview

Not all ideas are best expressed in words. Sometimes you need to sketch a diagram, draw a flowchart, map out a system architecture, or simply let your thoughts flow visually. Pointer's integrated whiteboard feature provides a powerful canvas for visual thinking, built on the open-source Excalidraw library that powers millions of diagrams worldwide.

The whiteboard in Pointer is not just an add-onâ€”it's a first-class feature designed to complement your text-based notes with visual content. Whether you're brainstorming ideas, explaining complex systems, creating wireframes, or just doodling to spark creativity, the whiteboard gives you a flexible, intuitive space to work visually.

## Getting started with whiteboards

### Accessing your whiteboard

Each user has a personal whiteboard that persists across sessions:

<Steps>
  <Step title="Navigate to Whiteboard">
    Click on the Whiteboard option in the main navigation menu or sidebar. This takes you to your personal whiteboard canvas.
  </Step>
  
  <Step title="Start creating">
    Your whiteboard opens with a blank canvas ready for use. The drawing tools are immediately available in the left toolbar.
  </Step>
  
  <Step title="Auto-save">
    Everything you draw is automatically saved. You can close your browser and return later to find your work exactly as you left it.
  </Step>
</Steps>

### The Excalidraw interface

The whiteboard uses Excalidraw's intuitive interface, which consists of:

- **Left toolbar** - Drawing tools and shape selection
- **Canvas** - The main drawing area (infinite, zoomable, pannable)
- **Top bar** - File operations, view controls, and collaboration settings
- **Right panel** - Element customization options (when an element is selected)

## Drawing tools

Excalidraw provides a comprehensive set of drawing tools for creating virtually any kind of visual content:

### Shape tools

| Tool | Shortcut | Description |
|------|----------|-------------|
| Selection | `V` or `1` | Select, move, and resize elements |
| Rectangle | `R` or `2` | Draw rectangles and squares |
| Diamond | `D` or `3` | Draw diamond/rhombus shapes |
| Ellipse | `O` or `4` | Draw circles and ellipses |
| Arrow | `A` or `5` | Draw arrows with customizable endpoints |
| Line | `L` or `6` | Draw straight lines |
| Pen | `P` or `7` | Freehand drawing tool |
| Text | `T` or `8` | Add text labels |
| Image | `I` | Insert images from your device |

### Drawing techniques

<Tabs>
  <Tab title="Basic shapes">
    Click and drag to create shapes:
    
    1. Select a shape tool from the toolbar
    2. Click where you want the shape to start
    3. Drag to define the size
    4. Release to complete the shape
    
    **Tips:**
    - Hold `Shift` while dragging to create perfect squares/circles
    - Hold `Alt` to draw from the center
  </Tab>
  
  <Tab title="Arrows and lines">
    Create connected diagrams:
    
    1. Select the Arrow or Line tool
    2. Click to start the line/arrow
    3. Click additional points to create bends
    4. Double-click or press `Enter` to finish
    
    **Tips:**
    - Arrows automatically snap to shape edges when connecting
    - Hold `Shift` to constrain to 45-degree angles
  </Tab>
  
  <Tab title="Freehand drawing">
    For sketchy, organic content:
    
    1. Select the Pen tool
    2. Click and drag to draw
    3. Release to complete the stroke
    
    **Tips:**
    - Use for annotations and emphasis
    - Great for brainstorming and ideation
    - Adjust stroke width in the right panel
  </Tab>
  
  <Tab title="Text">
    Add labels and descriptions:
    
    1. Select the Text tool
    2. Click where you want text
    3. Start typing
    4. Click outside or press `Escape` to finish
    
    **Tips:**
    - Double-click any text to edit it
    - Use text inside shapes for labeled diagrams
    - Adjust font size and style in the right panel
  </Tab>
</Tabs>

## Customizing elements

When you select an element, the right panel shows customization options:

### Stroke and fill

| Property | Options |
|----------|---------|
| Stroke color | Color picker with presets and custom colors |
| Background | Fill color (including transparent) |
| Stroke width | Thin, bold, extra bold |
| Stroke style | Solid, dashed, dotted |
| Fill style | Solid, hachure, cross-hatch, dots |

### Shape-specific options

| Shape | Additional options |
|-------|-------------------|
| Rectangles | Corner radius (rounded corners) |
| Arrows | Arrowhead style (both ends) |
| Lines | Endpoint styles |
| Text | Font family, size, alignment |

### Edge style

Excalidraw supports three edge styles that affect the overall look:

- **Sharp** - Clean, precise lines for technical diagrams
- **Round** - Softer corners for a friendlier look
- **Architect** - Hand-drawn, sketchy appearance for informal diagrams

## Canvas navigation

The whiteboard canvas is infinite and zoomable, allowing you to create diagrams of any size:

### Zooming

| Action | Method |
|--------|--------|
| Zoom in | `Ctrl/Cmd + +` or scroll wheel up |
| Zoom out | `Ctrl/Cmd + -` or scroll wheel down |
| Zoom to fit | `Shift + 1` |
| Zoom to selection | `Shift + 2` |
| Reset zoom | `Ctrl/Cmd + 0` |

### Panning

| Action | Method |
|--------|--------|
| Pan | Middle mouse drag, or `Space + drag` |
| Pan (trackpad) | Two-finger scroll |
| Pan (touchscreen) | Two-finger drag |

### View controls

- **Zoom controls** - Zoom in/out buttons in the bottom-left
- **Zoom indicator** - Shows current zoom percentage
- **Minimap** - Overview of the entire canvas (toggle in settings)

## Working with elements

### Selecting elements

| Action | Method |
|--------|--------|
| Select one | Click on element |
| Select multiple | `Shift + click` each element |
| Select area | Click and drag with Selection tool |
| Select all | `Ctrl/Cmd + A` |
| Deselect | `Escape` or click empty canvas |

### Manipulating elements

| Action | Method |
|--------|--------|
| Move | Drag selected elements |
| Resize | Drag corner handles |
| Rotate | Drag rotation handle (top center) |
| Duplicate | `Ctrl/Cmd + D` or `Alt + drag` |
| Delete | `Delete` or `Backspace` |

### Arranging elements

| Action | Method |
|--------|--------|

| Send backward | `Ctrl/Cmd + [` |
| Group | `Ctrl/Cmd + G` |
| Ungroup | `Ctrl/Cmd + Shift + G` |

## Whiteboard persistence

Pointer automatically manages whiteboard persistence so you never lose your work:

### How it works

1. Your whiteboard is stored as serialized Excalidraw data in Convex
2. Changes are auto-saved with debouncing (500ms after changes stop)
3. The full state is preserved including:
   - All elements and their properties
   - Canvas view state (zoom, position)
   - Selected elements
   - Collaboration state

### Data structure

The whiteboard is stored with the following fields:

```typescript
{
  title: string,           // Whiteboard name
  tenantId: string,        // Owner's user ID
  serializedData: string,  // Full Excalidraw state as JSON
  lastModified: string     // Timestamp of last change
}
```

## Tips and best practices

<Tip>
  **Use layers effectively:** Place background elements first, then build up layers. Use "Send to back" and "Bring to front" to manage element ordering.
</Tip>

<Tip>
  **Color coding:** Use consistent colors for different types of elements (e.g., blue for services, green for databases, red for warnings).
</Tip>

<Tip>
  **Text labels:** Add text labels to shapes to make diagrams self-documenting. Use the text tool or click inside shapes to add embedded text.
</Tip>

<Tip>
  **Templates:** Create common elements once, then duplicate them (`Ctrl/Cmd + D`) to build consistent diagrams quickly.
</Tip>

<Tip>
  **Export options:** Excalidraw supports export to PNG, SVG, and JSON formats for sharing diagrams outside of Pointer.
</Tip>
