## Welcome to the projects portfolio

This documentation provides detailed information about the various software engineering projects developed by Ryan Nguyen (wache online). These projects demonstrate expertise across full-stack development, AI integration, desktop application development, and developer tooling.

The projects are organized into several categories based on their primary functionality and purpose. Each project showcases different technical skills and approaches to solving real-world problems in software development.
