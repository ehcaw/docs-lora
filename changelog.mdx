## February 27, 2026

### Delta release - Portfolio infrastructure overhaul

This release introduces the complete Next.js portfolio application with an interactive IDE-style interface.

**What changed:**
- Added Next.js 14 application with App Router
- Implemented IDE-style layout with file tree navigation and markdown rendering
- Added project showcase images for Catdoc, Dashi, LSClear, Shelly, and Splat
- Created reusable UI components including file tree, image modal, and markdown renderer
- Set up Tailwind CSS with custom theme configuration
- Added portfolio data structure for experiences and projects

**Why it matters:**
The new portfolio design provides an intuitive developer-focused experience, allowing visitors to explore projects and experience through a familiar code editor interface.
