# Changelog

## January 2026

### Pointer: Next.js upgrade and comprehensive testing

**January 4, 2026**

A major update to Pointer bringing framework upgrades and a full test suite.

<Accordion title="What changed">
  - **Framework upgrade** - Upgraded Next.js to the latest version for improved performance and developer experience
  - **Comprehensive test coverage** - Added unit tests across core functionality including:
    - Collaborative editor utilities
    - URL metadata fetching
    - Whiteboard utilities
    - Tree utilities and note management
    - Convex backend functions (note versions, task management, whiteboards)
  - **Vitest configuration** - Set up Vitest for both frontend and Convex backend testing
  - **CI improvements** - Added ESLint configuration and pull request templates
</Accordion>

<Accordion title="Why it matters">
  This update improves the stability and maintainability of Pointer. The test suite ensures that core features like collaborative editing, whiteboard functionality, and data management work reliably. The Next.js upgrade brings performance improvements and access to the latest React features.
</Accordion>
